# Doctor Workflow Guide

This guide outlines the primary workflow for a Doctor using the Ghana EMR system.

## 1. View Consultation Queue

-   Navigate to the "Consultation Queue" from the main dashboard.
-   Here you will see a list of patients who have been through Triage and are waiting for a consultation.

## 2. Start a Consultation

-   Select a patient from the queue to begin the consultation.
-   This will open the Consultation Form.

## 3. Fill Out Consultation Form

-   **Chief Complaint:** Record the patient's primary reason for the visit.
-   **Diagnoses:** Add one or more diagnoses using the ICD-10 autocomplete search.
-   **Prescriptions:** Add any necessary medications.
-   **Lab Orders:** Order any required lab tests.

## 4. End Consultation

-   Upon saving the form, the patient will be automatically moved to the next appropriate queue (e.g., Pharmacy or Billing).
