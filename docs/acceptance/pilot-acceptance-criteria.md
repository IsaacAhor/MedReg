# Pilot Acceptance Criteria

Technical
- NHIE sync success rate > 95% (Patient, Encounter/Obs).
- Eligibility response < 3 seconds (avg) with cache.
- Queue survives 72h outage; no duplicate submissions post-replay.

Security
- Privileges enforced; audit entries recorded for sensitive actions.

Adoption
- User satisfaction > 80%; daily active users > 90% of trained staff.
